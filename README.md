# 🧮 Calculator Web Application

This is a **web-based calculator project** developed using **HTML, CSS, and JavaScript** as part of my **CodeAlpha Internship Program**.  

The main goal of this project was to strengthen my understanding of **frontend development fundamentals**, including DOM manipulation, event handling, validation logic, and building interactive web applications from scratch.

---

## 📌 Project Overview

The Calculator Web App allows users to perform basic arithmetic operations through both **mouse clicks** and **keyboard input**.  

This project was built step-by-step to understand how real-world frontend applications manage:

- User input
- Internal state management
- Validation logic
- Dynamic UI updates

Instead of relying only on built-in shortcuts, the logic was developed to improve problem-solving and JavaScript fundamentals.

---

## 🚀 Features

### ✔️ Arithmetic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

### ✔️ Input Handling
- Number buttons (0–9)
- Operator buttons
- Decimal support (.)
- Expression display

### ✔️ Validation Logic
- Prevents multiple operators in sequence
- Prevents invalid starting operators
- Prevents multiple decimals in one number

### ✔️ Utility Functions
- Clear (C)
- Backspace (⌫)
- Continuous calculation support

### ✔️ Keyboard Support
- Numbers (0–9)
- Operators (+, -, *, /, .)
- Enter key → Calculate
- Backspace key → Delete last value
- Escape key → Clear display

### ✔️ Controlled Display
- Display is **read-only** to prevent manual typing
- All inputs go through JavaScript logic
